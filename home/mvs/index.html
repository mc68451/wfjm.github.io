Overview

The IBM MVS releases up to 3.8j (24-bit, released in 1981) are freely available and can be run with an IBM System/370 emulator like Hercules. Current MVS related GitHub projects are

  • mvs38j-langtest: MVS 3.8J Compiler and Language tests
  • herc-tools: Tools for Hercules and MVS 3.8J Turnkey Systems
  • s370-perf: IBM System/370 Instruction Timing Benchmark
  • mvs38j-sios: Simple I/O System for MVS 3.8J Assembler Tests
